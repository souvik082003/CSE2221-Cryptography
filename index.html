Cryptography Interactive Lab

A specialized visualization toolkit for CSE students

Vigenère Cipher

Implement polyalphabetic substitution using keyword-based shifts and a Vigenère table.

Hill Cipher

Visualize polygraphic substitution based on linear algebra and modular matrix multiplication.

Playfair Cipher

Simulate digraph encryption using a 5x5 character matrix and classic rectangle rules.

Rail Fence

Observe transposition patterns by rearranging characters in a zig-zag "fence" structure.

Cryptanalysis

Test various attack vectors including Brute Force and Frequency Analysis on common ciphers.
